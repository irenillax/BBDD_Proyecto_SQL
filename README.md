Proyecto SQL:
- El objetivo de este proyecto ha sido aplicar consultas SQL sobre una base de datos relacional en PostgresSQL, utilizando la herramienta DBeaver.
- Desarrollar un análisis completo sobre una base de datos relacional. 

Contenido del repositorio: 
- *Consultas_Proyecto_sql: Contiene las 64 consultas SQL desarrolladas durante el proyecto. Cada consulta está numerada e incluye comentario/nota en algunas de las consultas.
- *Esquema Entidad Relación BBDD: Diagrama ER exportado desde DBeaver que muestra las tablas y sus relaciones principales.
- README: Documento explicativo con los pasos, objetivos y herramientas utilizadas.

Contenido de la Base de Datos: 
La base de datos utilizada representa un sistema de alquiler de películas con las siguientes tablas principales: film, actor, category, inventory, rental, payment, store, customer y staff. El esquema entidad-relación incluido muestra las relaciones entre las tablas (por ejemplo, las claves foráneas entre film_actor, film_category o rental con payment).

Consultas destacadas. A lo largo del proyecto se han abordado diferentes tipos de consultas: 
- Búsquedas simples y filtradas (WHERE,BETWEEN).
- Uniones entre tablas (INNER, JOIN, LEFT JOIN).
- Funciones de agregación (COUNT, AVG, SUM, MAX, MIN).
- Agrupaciones con GROUP BY y filtrado con HAVING.
- Creación de vistas y tablas temporales.
- Consultas anidadas y subconsultas. 

Cómo ejecutar las consultas: 
1. Abrir DBeaver y conectarse a la base de datos 'BBDD_Proyecto'.
2. Cargar el archivo 'Consultas_proyecto.sql'.
3. Ejecutar las consultas una por una para visualizar los resultados.
4. Consultar el archivo 'Esquema Entidad Relación BBDD para realizar este proyecto.

Herramientas utilizadas:
1. DBeaver -> Entorno de trabajo SQL.
2. PostgresSQL -> Motor de base de datos.

Resultados y conclusiones:
- Comprender la estructura y relaciones internas de una base de datos relacional.
- Desarrollar consultas SQL de diferente nivel de complejidad.
- Aplicar buenas prácticas de documentación en código y versionado de proyectos con GitHub.
- Generar una base sólida para el manejo avanzado de SQL en contextos analíticos y de negocio.
